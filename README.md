# 日本語単語練習ゲーム 🎌

JLPT N5〜N1レベルの日本語単語を楽しく学べるゲームです。

## 機能

### 🌍 多言語対応
- **英語** (English)
- **中国語** (中文)
- **ベトナム語** (Tiếng Việt)
- **アラビア語** (العربية)
- **スペイン語** (Español)
- **ドイツ語** (Deutsch)
- **フランス語** (Français)

### 📚 レベル
- **N5** - 初級レベル（718単語）
- **N4** - 初中級レベル（668単語）
- **N3** - 中級レベル（2,139単語）
- **N2** - 中上級レベル（1,748単語）
- **N1** - 上級レベル（2,699単語）
- **合計：7,972単語**

### 🎮 ゲームモード
1. **フラッシュカード** - 単語を見て、クリックで答えを確認
2. **多肢選択** - 4つの選択肢から正しい意味を選ぶ
3. **タイピング** - 読み方を入力して答える

### ✨ 機能
- 🌍 7言語対応の意味表示
- 🎨 美しいアニメーション効果
- ✨ パーティクル爆発エフェクト
- 🔥 コンボシステム
- 📊 リアルタイムスコア表示
- 💯 連続正解でボーナスポイント
- 📈 進捗バー
- 📱 レスポンシブデザイン

## 使い方

1. ブラウザで `index.html` を開く
2. JLPT レベル（N5〜N1）を選択
3. ゲームモードを選択
4. 楽しく練習！

## 単語の追加・編集方法

### JSONファイルの形式

各レベルの単語は `vocabulary/` フォルダ内のJSONファイルに保存されています。

**フォーマット：**
```json
[
  {
    "word": "学生",
    "reading": "がくせい",
    "meaning": "student"
  },
  {
    "word": "先生",
    "reading": "せんせい",
    "meaning": "teacher"
  }
]
```

### 単語を追加する手順

1. 対応するレベルのJSONファイルを開く（例：`vocabulary/N5.json`）
2. 配列の最後に新しい単語オブジェクトを追加：
```json
{
  "word": "新しい単語",
  "reading": "あたらしいたんご",
  "meaning": "new word"
}
```
3. カンマの位置に注意してください：
   - 配列内の各オブジェクトの間にはカンマが必要
   - 最後のオブジェクトの後にはカンマは不要

### 例：単語を追加

**変更前：**
```json
[
  { "word": "学生", "reading": "がくせい", "meaning": "student" },
  { "word": "先生", "reading": "せんせい", "meaning": "teacher" }
]
```

**変更後：**
```json
[
  { "word": "学生", "reading": "がくせい", "meaning": "student" },
  { "word": "先生", "reading": "せんせい", "meaning": "teacher" },
  { "word": "友達", "reading": "ともだち", "meaning": "friend" }
]
```

### 💡 ヒント

- **大量の単語を追加する場合：**
  - Excelやスプレッドシートで作成
  - オンラインのCSV to JSONコンバーターを使用
  - JSONファイルに貼り付け

- **単語リストを探す：**
  - JLPT公式単語リスト
  - 日本語学習サイト
  - 単語帳アプリのデータ

- **バックアップ：**
  - 編集前に必ずバックアップを取る
  - JSONの形式が正しいか確認（https://jsonlint.com/）

## 単語を更新・追加する方法

### 方法1: GitHubから最新の単語をダウンロード（推奨）

最も簡単な方法です！ワンクリックで全レベルの単語を更新：

```bash
python download_vocabulary.py
```

このスクリプトは：
- GitHub（https://github.com/elzup/jlpt-word-list）から最新の単語を取得
- 自動的にJSON形式に変換
- 既存のファイルを上書き
- 合計7,972単語以上をダウンロード

### 方法2: CSVファイルから変換

自分で作成したCSVファイルを変換する場合：

#### 1. CSVファイルを準備
```csv
word,reading,meaning
学生,がくせい,student
先生,せんせい,teacher
```

#### 2. 変換スクリプトを実行
```bash
python convert_csv_to_json.py your_file.csv vocabulary/N5.json
```

#### 3. 手動変換（オンラインツール）
- https://csvjson.com/csv2json
- https://www.convertcsv.com/csv-to-json.htm

## トラブルシューティング

### 問題：単語が読み込まれない
- JSONファイルの形式が正しいか確認
- ファイル名が正しいか確認（N5.json, N4.json など）
- ブラウザの開発者ツール（F12）でエラーを確認

### 問題：単語が表示されない
- ブラウザのキャッシュをクリア（Ctrl+F5）
- ファイルのエンコーディングがUTF-8であることを確認
